# 2. 주소창에 입력했을 때 찾아가는 과정

1. 주소를 입력
2. 로컬에 등록된 DNS IP 조소로 해당 도메인을 질의 (query) 함
   1. 해당 DNS에 정보가 캐싱된 경우는 바로 응답받음
   2. 없는 경우 상위 DNS 에게 질의
3. 응답받은 IP 주소로 접속(브라우저가)
4. 해당 서버에게 리소스 요청
